# Documentação do App de Chat - Desenvolvimento de Sistemas

## Criação de um app de chat utilizando Expo e react Native.

## Tecnonlogias usadas:

- EXPO
- React Native
- Firebase
- NativeWind
- Figma
- Authenticação com Firebase
- Mensagens Firebase
- Responsividade
- TailwindCSS (Nativewind 4)
- Lottie Animations
- Caching de imagens
- Cross-Platform Compatibility
